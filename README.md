# Athan - Prayer Times for Riyadh

A clean, minimal Arabic website that displays accurate prayer times for Riyadh, Saudi Arabia.

## Features

- **5 Prayer Times**: Fajr, Dhuhr, Asr, Maghrib, Isha
- **Smart Timer**: Countdown before Adhan, 30-min countup after Adhan
- **Hijri Date**: Accurate Kuwaiti algorithm (Umm Al-Qura based)
- **Bilingual**: Arabic / English toggle
- **Time Format**: 24h / 12h (AM/PM) toggle
- **Responsive**: Works on desktop, tablet, and mobile
- **No Dependencies**: Pure HTML, CSS, JavaScript

- HTML5
- CSS3 (Flexbox, Grid)
- Vanilla JavaScript
- [Aladhan API](https://aladhan.com/prayer-times-api) for prayer times

## Getting Started

1. Clone the repo:
   ```bash
   git clone https://github.com/yourusername/athan.git
   ```

2. Open `index.html` in your browser

No build step required.

## GitHub Pages Deployment

1. Push to GitHub
2. Go to **Settings > Pages**
3. Select **main branch** as source
4. Your site will be live at `https://yourusername.github.io/athan/`

## Calculation Method

Uses **Umm Al-Qura** calculation method, which is the standard for Saudi Arabia.
